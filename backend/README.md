# Backend

## initial setup
```
virtualenv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```


## How to run development
```
python main.py
```

## update requirements
```
pip freeze > requirements.txt
```

## How to load knowledge base
```
python load_knowledgebase.py
```

## How to run evaluate LLM
```
python evaluate_llm.py
```
